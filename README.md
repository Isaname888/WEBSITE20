# NexaWeb — WorldSkills Landing Page

Готовый проект для задания WorldSkills Web Technologies Frontend.

## Структура проекта

```text
worldskills_landing_max_score/
├── index.html
├── css/
│   └── style.css
└── images/
```

## Как открыть

1. Распакуй архив.
2. Открой `index.html` в браузере.
3. Или открой папку в VS Code и запусти через Live Server.

## Как загрузить на GitHub Pages

Важно: не загружай ZIP как один файл. Сначала распакуй архив.

В репозиторий нужно загрузить:
- `index.html`
- папку `css`
- папку `images`

`index.html` должен лежать в корне репозитория.

Путь к CSS уже правильный:

```html
<link rel="stylesheet" href="./css/style.css">
```

Поэтому после загрузки на GitHub Pages стили будут работать.
